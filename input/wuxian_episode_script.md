# 《雾线》分集剧本

# 第 1 集《浮尸》

## 本集梗概

林遥回到白雾镇为母亲办后事。旧水库枯水，村民发现一具包裹严密的尸骨。陈序初步判断尸体死亡时间超过十年。林遥在母亲遗物里发现一个没有寄出的快递，收件人是“周澄”。周澄明明是十二年前校车事故的遇难学生。

## 主要场景

### 1. 外景 / 旧水库 / 清晨

雾很厚，水位退到石阶下方。两个钓鱼老人发现黑色塑料布缠在枯木上。塑料布内露出一截手骨，骨节上卡着一枚生锈的工作牌。

老人甲说：“这水库吃过人，不奇怪。”  
老人乙盯着工作牌：“可这个人，像是被人包好才扔下去的。”

远处传来警笛，雾把声音压得很低。

### 2. 内景 / 长途车 / 上午

林遥坐在返乡大巴后排。手机里是母亲林婉的遗照。她收到殡仪馆电话，要求确认火化时间。她说“明天”，停顿后又改成“后天”。

旁边乘客问：“回白雾镇啊？那地方现在还起雾吗？”  
林遥：“一直起。”

### 3. 外景 / 林家老屋 / 中午

老屋院门半开，白布挂在门楣。邻居们低声议论，提到林婉去世前几天常去邮局。林遥进屋，看见母亲的房间被收拾得异常干净，桌上只有一只锁住的铁盒。

### 4. 内景 / 派出所临时勘验室 / 下午

陈序查看尸骨和随身物品。工作牌上字迹模糊，只剩“罗”“施工”几个字。法医说死亡时间至少十年以上，死因暂不能确定，但肋骨有锐器痕迹。陈序要求调十二年前旧水库周边失踪人口。

同事说：“十二年前这地方最大的案子就是校车坠湖。死了十七个学生，一个司机。”  
陈序没有接话，看向墙上旧地图。

### 5. 内景 / 林家老屋 / 夜

林遥打开铁盒，里面有旧账册、录音带、一个没寄出的牛皮纸快递。快递单收件人写着：周澄。地址是“白雾镇云台中学旧宿舍”。

林遥拨给周砚。电话接通后，她只问一句：“你弟弟的名字，为什么会出现在我妈的快递上？”

周砚沉默很久：“你别拆。你在哪？我现在过去。”

### 6. 外景 / 云台中学旧址 / 夜

周砚赶到。学校旧址荒废多年，楼道里积水。林遥坚持拆快递，里面是一张旧照片：校车停在分岔路口，车窗里有孩子的手贴在玻璃上；照片背后写着“车不是从这里掉下去的”。

周砚说：“我找了十二年，所有人都说我疯了。”  
林遥：“我妈为什么不直接给你？”  
周砚：“因为她怕。”

### 7. 内景 / 陈序办公室 / 夜

DNA 初筛结果回传，尸骨并非校车遇难学生，而是失踪施工员罗志强。陈序翻出旧卷宗，发现罗志强失踪日期正是校车事故后三天。

陈序看到卷宗封面上父亲陈国良的签名。镜头停在签名上。

### 8. 外景 / 林家老屋 / 深夜

林遥回家，院门口多了一个黑色塑料袋。里面是一卷湿透的磁带和一张纸条：“别让周澄回来。”

林遥抬头，巷子尽头有人影闪过。

## 本集结尾钩子

林遥用旧录音机播放磁带，里面传来母亲林婉年轻时的声音：“他们说车上十七个孩子都死了。可是我听见了第十八个孩子在哭。”

# 第 2 集《第十八个孩子》

## 本集梗概

林遥、周砚试图确认“第十八个孩子”是谁。陈序调查罗志强失踪，发现罗志强曾参与云台新路施工。旧卷宗显示，当年校车核载人数、上车名单、遇难名单存在细微差异。

## 主要场景

### 1. 内景 / 林家老屋 / 清晨

林遥反复听录音。磁带受潮严重，只能听见几个断裂词：“新路”“孩子”“账”“不能签”。周砚坐在一旁，手里拿着弟弟周澄的旧学生证。

周砚说：“周澄那天坐在最后一排。他晕车，喜欢靠窗。”  
林遥：“如果他没死，为什么十二年不回来？”  
周砚：“也许他回不来。”

### 2. 内景 / 县公安局档案室 / 上午

陈序查旧档。校车事故官方名单显示：学生十七人，司机一人，随车教师无。另一份学校考勤表却显示当日有十八名学生报名参加县里竞赛。第十八人的名字被墨水涂黑。

档案员提醒：“陈队，这案子当年结得很快。你父亲那会儿压力也大。”  
陈序：“压力大，不等于可以少一页。”

### 3. 外景 / 修车铺 / 中午

陈序找许嵩。许嵩对警察有敌意，认为警方把所有责任推给他父亲。墙上挂着司机许立群的旧照，照片被烟熏黑。

许嵩说：“我爸开了二十年山路，从没出过事。他不是困，他是被人逼着走那条路。”

陈序问谁逼他。许嵩指向镇政府旧楼方向：“你们都知道。”

### 4. 内景 / 周家 / 下午

周砚给林遥看他这些年收集的资料：事故报道、家属联名信、被退回的申诉材料。林遥发现一张照片中，救援现场有一个戴蓝色雨衣的孩子坐在担架边缘，但脸被记者水印挡住。

林遥：“这个孩子是谁？”  
周砚：“我不知道。新闻发出三小时后，这张图就被删了。”

### 5. 内景 / 县医院档案室 / 傍晚

林遥找沈禾。沈禾起初否认知道内情。林遥拿出母亲留下的快递，沈禾脸色变了。她说当年夜里医院接收过一名“无名少年”，但第二天系统里记录被改成“误录”。

沈禾：“我只看见他脖子上有一个红色线绳，挂着半块平安扣。”

### 6. 外景 / 白雾镇老街 / 夜

林遥和周砚走过老街。有人在暗处拍照。周砚追上去，对方逃进巷子。地上掉下一张快递底单，寄件人姓名空白，寄件地址是“云台新路项目部旧址”。

### 7. 内景 / 陈序家 / 夜

陈序回家翻父亲遗物。父亲已瘫痪多年，无法清楚表达。陈序在父亲旧书中找到一张便利贴：罗志强——不能按失踪办。

陈序问父亲：“为什么？”  
父亲眼角抖动，嘴里含混地说：“水……上面……有人……”

## 本集结尾钩子

林遥将照片放大，担架边的蓝雨衣少年露出半张侧脸。周砚看清后崩溃：那个人很像周澄。

# 第 3 集《新路》

## 本集梗概

调查转向云台新路工程。林遥发现母亲曾为学校经手一笔“临时改线补偿款”。陈序查到罗志强失踪前曾举报新路边坡未加固。顾明川第一次正式登场，试图以“镇上安宁”为由阻止重查。

## 主要场景

### 1. 外景 / 云台新路旧址 / 清晨

新路早已废弃，半截护栏悬在塌方边缘。路边杂草覆盖一块旧警示牌：“施工区域，禁止通行”。林遥拍摄路面裂缝，发现水泥下有被烧过的轮胎印。

周砚说：“官方说校车走老路，雨太大冲进水库。”  
林遥：“可这条路离水库更近。”

### 2. 内景 / 镇政府档案室 / 上午

林遥以拍纪录片为由查地方志。她看见云台新路竣工时间写在事故前三天，但工程照片明显是晴天补拍。管理员接到电话后，要求她立刻离开。

### 3. 内景 / 县公安局 / 上午

陈序查工程举报记录。罗志强曾提交边坡加固问题，但举报件被转为“施工纠纷”。签批人是顾明川。

同事说：“顾明川现在是县政协委员，碰他要证据。”  
陈序：“所以我在找。”

### 4. 内景 / 林家老屋 / 下午

林遥翻母亲账册，发现事故前一天学校支出一笔 4800 元“车辆临时改线协调费”。收款人没有姓名，只有代号“G”。

林遥意识到母亲当年知道校车改线。

### 5. 外景 / 白雾镇茶馆 / 下午

顾明川约见林遥。他对林婉去世表示慰问，语气温和却处处压迫。

顾明川：“你母亲一辈子求稳。她最不想看见的，就是你把旧事翻出来，让活着的人继续受罪。”  
林遥：“活着的人里面，也包括你吗？”

顾明川微笑：“包括整个白雾镇。”

### 6. 内景 / 修车铺 / 傍晚

许嵩告诉陈序，当年父亲接到临时通知，必须改走新路，因为老路“有领导车队经过”。但他找不到通话记录，手机早被警方收走。

陈序问：“你为什么现在才说？”  
许嵩：“我说过。你们没写。”

### 7. 外景 / 云台新路 / 夜

林遥独自回到新路拍摄。雾中突然亮起车灯，一辆皮卡朝她冲来。她躲进排水沟，摄影机摔在路边。车停下，有人捡起摄影机，删除存储卡后离开。

### 8. 内景 / 林家老屋 / 深夜

林遥清理伤口，发现母亲账册夹层里有一张收据复印件，盖章单位是“白雾镇云台新路项目指挥部”，经手人：罗志强。

## 本集结尾钩子

陈序接到法医电话：罗志强肋骨上的伤不是坠落造成，而是死前被尖锐物刺入。罗志强是被杀的。

# 第 4 集《误录》

## 本集梗概

医院“误录”的无名少年成为关键。沈禾承认当年有孩子被送来，但很快被人带走。林遥发现母亲曾深夜到医院签过一份证明。周砚认为弟弟可能活着，行动开始失控。

## 主要场景

### 1. 内景 / 县医院 / 上午

沈禾带林遥进入旧病案库。纸质病历因水浸霉斑严重。沈禾找到 2014 年暴雨夜急诊登记簿，其中一行被刀片刮掉，只留下“男，约十五岁，溺水，意识模糊”。

沈禾：“第二天早上，登记系统里就没有这个人了。”  
林遥：“谁删的？”  
沈禾：“医院删不了公安系统。公安系统也删不了医院纸档。能让两边都闭嘴的人，不在医院。”

### 2. 外景 / 周家院子 / 中午

周砚看到蓝雨衣少年照片后情绪崩溃。他开始逐户询问当年救援队员，方式粗暴，引发冲突。一位老人被逼急后说：“你弟要是活着，也早被你们找死了。”

### 3. 内景 / 陈序办公室 / 下午

陈序发现当年值班医生三个月后调离本地，后来因车祸去世。死者妻子保存了一只纸箱，里面有一份未完成的病程记录复写纸，写着“无名少年疑似醒后反复说：不是水，是石头”。

### 4. 内景 / 林家老屋 / 黄昏

林遥在账册里看到母亲字迹：“证明已签，欠周家。”她无法理解母亲为什么会帮忙掩盖。她回忆事故当晚，自己高烧躺在床上，母亲守在床边，外面警笛不断。

小林遥：“他们为什么哭？”  
林婉：“山上有车掉水里了。”  
小林遥：“周澄呢？”  
林婉没有回答。

### 5. 外景 / 白雾镇客运站旧址 / 夜

周砚收到匿名短信：“想知道你弟去哪，来旧客运站。”他独自赴约，被人打晕。醒来时，身边放着一枚半块平安扣。

### 6. 内景 / 顾明川家 / 夜

顾明川接到电话，对方说周砚拿到了平安扣。顾明川沉默片刻，只说：“让他相信周澄还活着，比让他相信周澄死了更容易控制。”

### 7. 内景 / 陈序家 / 深夜

陈序父亲突然清醒片刻。他抓住陈序手腕，说：“孩子不是我放走的……我只是没追。”陈序追问是谁，父亲又陷入混乱。

## 本集结尾钩子

林遥打开母亲留下的第二卷录音。林婉声音颤抖：“我签了那张证明。证明周澄已经死亡。可我知道，他当时还有呼吸。”

# 第 5 集《半块平安扣》

## 本集梗概

周砚认定周澄活着，追查平安扣来源。陈序查到罗志强妻子当年收到过一笔匿名汇款。林遥开始怀疑母亲参与程度更深。案件进入“生者身份”线，但多方信息互相矛盾。

## 主要场景

### 1. 内景 / 周家 / 清晨

周砚把半块平安扣放在旧照片旁。照片里周澄脖子上挂着完整平安扣。周母看到后病情加重，反复问“他是不是回来了”。周砚决定去找当年给周澄做平安扣的玉器店。

### 2. 外景 / 县城玉器街 / 上午

玉器店老板记得这块扣，说事故后有人拿另一半来修过，但没有修成。付款人留下的电话号码早已停机，名字登记为“罗”。

周砚：“罗志强？”  
老板：“也可能姓骆、落、洛，口音重，我记不清。”

### 3. 内景 / 县公安局 / 中午

陈序查到罗志强妻子在丈夫失踪后一个月收到 6 万元汇款，汇款账户来自外省空壳公司。公司后来并入顾明川亲戚名下的矿业公司。

同事建议并案重查。领导提醒：“旧案牵涉面太大，先别惊动县里。”

### 4. 内景 / 林家老屋 / 下午

林遥发现母亲账册后半部分有几页被撕掉。撕口处夹着一根红色线绳。她想起周澄照片里的红绳，意识到母亲可能接触过被救出的孩子。

### 5. 外景 / 废弃矿道入口 / 傍晚

许嵩找到林遥，说当年父亲出车前接过一张手写路线图，路线图来自项目部。许嵩突然改口，不愿把图交出，因为顾明川的人威胁他妻儿。

林遥：“你爸背了十二年骂名。”  
许嵩：“我爸已经死了。我儿子还要上学。”

### 6. 内景 / 顾明川办公室 / 夜

顾明川会见矿业公司律师，要求清理旧材料。律师提醒罗志强尸骨已经出现，风险无法完全切断。顾明川说：“死人不会开口。麻烦的是那些以为自己还活着的人。”

### 7. 外景 / 河堤 / 夜

周砚收到一通陌生电话。电话里是成年男性声音：“哥，别找了。”周砚呆住，追问对方是不是周澄。对方挂断。

周砚立刻把录音发给林遥。林遥听完，没有立刻说话。声音像周澄，但语气过于平稳。

## 本集结尾钩子

陈序拿到声纹初步比对：电话声纹与周澄少年时期公开视频无法确认同一人，但与罗志强儿子罗小满有高度相似。

---

# 第 6 集《活人证明》

## 本集梗概

“周澄电话”被证实可能是误导。罗志强儿子罗小满出现，称父亲曾救过一个学生，并把他藏在矿道工棚。顾明川推动舆论将调查包装成“家属借旧案炒作”。林遥和陈序出现信任裂痕。

## 主要场景

### 1. 内景 / 县公安局审讯室 / 上午

罗小满被带来。他承认给周砚打电话，但说是有人给他 2 万元让他这么做。他小时候听母亲提过，罗志强死前曾带回一个男孩，男孩发烧，说梦话喊“哥”。

陈序：“那孩子后来去哪了？”  
罗小满：“我妈说，被一个女人接走了。”

### 2. 内景 / 林家老屋 / 中午

林遥看到“女人”这个线索后沉默。她怀疑是母亲林婉接走周澄。周砚得知后愤怒，认为林遥一直在保护林婉。

周砚：“你妈知道我弟活着，却让我妈每年给空坟烧纸。”  
林遥：“我也刚知道。”  
周砚：“你们林家每一句刚知道，都刚好晚十二年。”

### 3. 外景 / 白雾镇广场 / 下午

本地自媒体发布视频，指责林遥借母亲去世炒作旧案，周砚精神偏执，陈序滥用警力。镜头切到顾明川接受采访，表示“理解家属痛苦，但白雾镇需要向前看”。

### 4. 内景 / 县公安局 / 下午

领导要求陈序暂停公开动作。陈序坚持罗志强命案已经成立。领导没有否认，只说：“命案可以查，十二年前的校车事故不要碰。”

### 5. 内景 / 沈禾家 / 傍晚

沈禾终于交出一张旧护士排班表。表背面写着一个地址：雾岭福利院。沈禾说，当年有人安排无名少年转走，手续上写的是“临时救助”。

### 6. 外景 / 雾岭福利院旧址 / 夜

林遥和陈序来到废弃福利院，找到一本残缺登记册。某页写着“无名，男，约十五岁，送来人：林婉”。离院日期在两个月后，接走人签名被撕掉。

林遥情绪崩溃：“所以她真的把他带走了。”  
陈序：“至少说明他活过。”

### 7. 内景 / 陈序车内 / 夜

林遥质问陈序是否早就怀疑她母亲。陈序承认没有告诉她部分线索，因为她是利害关系人。两人关系破裂。

### 8. 外景 / 福利院后院 / 深夜

林遥独自返回福利院，在墙缝里找到一张孩子刻下的字：“我叫周澄。我不想忘。”旁边还有一串数字：0617。

## 本集结尾钩子

陈序查到 0617 对应一份被封存的户籍变更记录：十二年前 6 月 17 日，一个名叫“程舟”的少年在邻县落户，监护人签字是顾明川。

---

# 第 7 集《程舟》

## 本集梗概

周澄改名“程舟”后曾在邻县生活，但三年前再次消失。林遥发现母亲一直在给程舟汇款。陈序确认顾明川当年不是救人，而是控制活口。周砚找到程舟留下的线索，冲动闯入顾明川控制的矿区。

## 主要场景

### 1. 外景 / 邻县老居民区 / 上午

林遥、陈序走访程舟旧住处。房东说程舟安静、怕水、从不拍照，三年前突然搬走，只留下一箱书。书里夹着白雾镇旧地图和矿区通行证复印件。

### 2. 内景 / 银行自助区 / 中午

陈序调取资金流水。林婉从事故后第二年起，每年向程舟账户汇款，金额不大但从未中断，直到去世前一个月。最后一笔备注是：“别回来。”

林遥站在屏幕前，第一次哭出声。

### 3. 内景 / 程舟旧屋 / 下午

林遥在箱底找到一本笔记。程舟写道：他记得坠湖前车被石头砸中，罗志强把他从后窗拖出来；他记得有人说“只少一个，不影响名单”；他记得一个女人抱着他说“你活着会害死更多人”。

### 4. 外景 / 白雾镇矿区门口 / 傍晚

周砚没有等警方，独自来矿区找顾明川。他用平安扣威胁门卫，声称自己知道程舟还活着。门卫放他进去后立刻锁门。

### 5. 内景 / 顾明川车内 / 傍晚

顾明川接到消息，表情第一次失控。他命令手下找到周砚拿走的东西，不要伤人，“至少不要在矿区”。这句话暴露他对舆论后果的恐惧。

### 6. 内景 / 矿区旧办公楼 / 夜

周砚在旧办公楼找到程舟留下的 U 盘。U 盘外贴着“如果我没出去，交给林婉”。他刚拔下 U 盘，楼道灯熄灭。有人追来。

### 7. 外景 / 矿区山路 / 夜

林遥和陈序赶到矿区外，听见远处传来撞击声。周砚满身是血跑出来，把 U 盘塞给林遥，却被身后车辆撞倒。车牌被泥遮住。

### 8. 内景 / 县医院抢救室外 / 深夜

周砚昏迷。林遥打开 U 盘，里面只有一个视频文件，画面是程舟成年后的自拍。

程舟说：“如果你们看到这个，说明我没能把账册拿出去。不要相信顾明川，也不要完全相信林婉。她救过我，也卖过我一次。”

## 本集结尾钩子

视频最后，程舟举起一页账册原件：“校车不是自己掉下去的。有人在塌方后，把还活着的人重新推回了水里。”

# 第 8 集《雾线》

## 本集梗概

程舟视频指向最后证据：原始账册和罗志强遇害现场记录藏在旧矿道。林遥、陈序进入矿道寻找证据。顾明川试图销毁矿道。陈序面对父亲当年沉默的真相。案件最终重启，但每个人都要承受迟来的代价。

## 主要场景

### 1. 内景 / 县公安局 / 清晨

陈序提交并案申请：罗志强命案、周澄失踪案、校车事故疑点。领导仍犹豫，直到周砚被撞的监控显示涉案车辆与矿业公司有关。专案组成立，但必须在证据灭失前行动。

### 2. 内景 / 陈序家 / 上午

陈序父亲看完程舟视频后崩溃。他终于说出当年真相：救援现场曾发现校车后窗破裂，有孩子逃生痕迹；他怀疑有人移动现场，但顾明川以“稳定家属、避免群体事件”为由压下。陈国良没有收钱，却选择服从。

陈序：“你不是不知道。”  
陈国良：“我知道得不够多。也怕知道得太多。”

### 3. 外景 / 旧矿道入口 / 下午

林遥根据程舟地图找到矿道暗门。陈序要求她留在外面，林遥拒绝。她说：“我妈把证据藏起来，轮到我把它拿出去。”

### 4. 内景 / 旧矿道 / 下午

矿道潮湿狭窄，墙上有程舟刻下的箭头。两人找到铁皮箱，里面有原始账册、路线图、罗志强手写记录、事故当夜现场照片。照片显示校车撞上塌方落石后停在路边，车体并未立刻坠湖。

罗志强记录：车上仍有三名学生有生命迹象。有人命令“清场”。

### 5. 内景 / 旧矿道深处 / 傍晚

顾明川带人出现。他承认当年新路工程存在重大隐患，但否认自己下令杀人。他把责任推给已死的施工队长，声称自己只是“处理残局”。

林遥：“罗志强也是残局？”  
顾明川：“他要把镇上所有人拖下水。”  
陈序：“你说的所有人，其实只有你自己。”

### 6. 内景 / 旧矿道 / 傍晚

矿道外传来爆破声，顾明川的人试图制造塌方毁证。混乱中，林遥被困在低洼处，水迅速上涨。她想起童年事故夜的警笛声，几乎窒息。陈序返回救她，铁皮箱被水冲走一半。

林遥抓住箱带：“我妈没能寄出去。”  
陈序：“那就别松手。”

### 7. 外景 / 旧矿道入口 / 夜

专案组赶到，顾明川被控制。林遥抱着铁皮箱出来，浑身是泥。周砚从医院醒来，收到林遥发去的照片：完整平安扣、账册、程舟留下的字。

### 8. 内景 / 审讯室 / 数日后

顾明川面对证据沉默。警方通报：罗志强被害案立案侦查，校车事故重新调查，相关责任人追责。程舟仍未找到，但他的证言视频被认定为重要线索。

### 9. 外景 / 白雾镇旧水库 / 清晨

林遥把母亲未寄出的快递重新封好，收件人仍写周澄，寄件人改成林遥。她把快递交给周砚。周砚没有拆，只说：“等他回来。”

### 10. 外景 / 云台新路 / 清晨

雾散开一瞬，露出断裂的新路。林遥举起摄影机拍摄。镜头里，路尽头站着一个戴帽子的男人，脖子上隐约有红绳。林遥放下摄影机，男人已不见。

## 全剧结尾

林遥的纪录片开头旁白响起：“白雾镇的人总说，雾大时不要赶路。因为你以为自己走的是路，其实只是别人替你留好的一条线。”

画面切黑。
