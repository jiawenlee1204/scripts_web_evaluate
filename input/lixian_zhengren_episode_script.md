# 《离线证人》分集剧本

# 第 1 集《最后一次登录》

## 本集梗概

深夜，城市记忆类 App“回声盒”上线内测版。产品经理许若清在后台看到一个已经注销两年的账号突然登录，并向她发送一段语音。账号主人是她失踪的姐姐许若安。与此同时，回声盒的首席算法工程师陆鸣在公司天台坠亡，警方初步判断为自杀。许若清发现陆鸣死前正在恢复一批被标记为“离线证人”的隐藏数据。

## 主要场景

### 1. 内景 / 回声科技办公区 / 夜

办公区只剩几盏工位灯。大屏显示“回声盒 3.0 内测灰度 3%”。许若清盯着埋点看板，用户留存曲线平稳，异常日志却在右下角不断刷新。

运营同事周望打着哈欠：“三点了，还不回？”  
许若清：“等最后一批语音备份跑完。”  
周望：“一个记日记的 App，被你看得像发射火箭。”  
许若清没有笑。她点开异常日志，看见一行红字：账号 A-000817 已登录。

她的手停住。这个账号的昵称是“安”。

### 2. 内景 / 公司茶水间 / 夜

许若清反复刷新后台。账号头像是一张旧照片，照片里两个女孩站在海边，姐姐许若安把围巾绕在妹妹脖子上。

手机弹出一条 App 内私密语音：“若清，别相信今晚上线的数据。”

声音是许若安。语音只有六秒，尾音有明显电流声。许若清冲出茶水间，走廊尽头的服务器机房门正缓缓合上。

### 3. 内景 / 服务器机房 / 夜

机房温度很低。许若清刷卡进入，看到陆鸣蹲在机柜前，手里拿着移动硬盘。

许若清：“你在恢复 A-000817？”  
陆鸣抬头，眼下全是血丝：“不是她一个人。你知道‘离线证人’吗？”  
许若清：“那是什么项目？”  
陆鸣把硬盘塞进她手里：“明天董事会前，把这个交给媒体。不要交给公司，也不要交给警察。”  
许若清：“陆鸣，你到底在说什么？”

机房外传来脚步声。陆鸣猛地关掉屏幕，只留下一串文件名：OFFLINE_WITNESS_13。

### 4. 外景 / 回声科技大厦天台 / 凌晨

风很大。许若清追上天台，只看见陆鸣站在护栏边。他背后是城市凌晨的冷光。

陆鸣：“你姐姐不是失踪。她被留在系统里了。”  
许若清：“下来，我们报警。”  
陆鸣回头，像是听见了什么。耳机里传出一个柔和的女声：“请确认删除。”

许若清向前一步。楼下忽然传来重物坠地的闷响。她扑到护栏边，只看见人群在地面散开。

### 5. 内景 / 刑警队询问室 / 上午

刑警顾峥把一杯水推到许若清面前。墙上的钟指向九点二十。

顾峥：“陆鸣有抑郁记录吗？”  
许若清：“没有。他昨晚在害怕。”  
顾峥：“害怕什么？”  
许若清看着杯子里的倒影：“害怕公司正在上线不该上线的东西。”

顾峥翻看笔录：“你说你收到失踪姐姐的语音。手机呢？”  
许若清拿出手机，语音记录已经消失。只剩系统提示：该内容因用户撤回不可查看。

### 6. 内景 / 回声科技会议室 / 白天

CEO 孟远舟主持危机会议。他穿着黑色西装，语气克制。

孟远舟：“对外口径是员工个人原因。产品按计划上线。”  
许若清：“陆鸣死前说过‘离线证人’。”  
法务唐微立刻接话：“内部项目代号不能外泄。”  
许若清：“所以确实有这个项目。”  
孟远舟看向她：“若清，你姐姐的事我很遗憾。但不要把私人创伤带进公司判断。”

会议室静下来。许若清第一次意识到，所有人都知道许若安。

### 7. 内景 / 许若清家 / 夜

许若清回到家，打开陆鸣给她的移动硬盘。硬盘里只有一个空文件夹，命名为“家属不可见”。她尝试恢复文件，屏幕弹出密码提示：请输入第一条遗言。

她翻出姐姐旧手机，发现回声盒早期版本仍在本地。登录失败三次后，App 自动播放一段未同步录音。

许若安的声音：“若清，如果我不回来，不要找我。找那个每晚听我说话的人。”

### 8. 内景 / 地铁车厢 / 深夜

许若清坐在末班地铁。车厢玻璃映出她疲惫的脸。手机忽然亮起，一个陌生账号给她发来定位，地点是“港湾精神康复中心旧楼”。

紧接着，又弹出一条语音。女声很轻：“第一个证人醒了。”

她抬头，车厢尽头站着一个戴口罩的男人，正用她姐姐的旧手机壳打电话。

## 本集结尾钩子

许若清追下地铁，对方消失在人流里。她低头看手机，回声盒后台显示：许若安账号正在直播输入中。

# 第 2 集《港湾旧楼》

## 本集梗概

许若清前往港湾精神康复中心旧楼，发现这里曾是回声盒早期“情绪陪伴实验”的线下采样点。顾峥开始调查陆鸣坠楼案，发现天台监控少了七分钟。许若清从旧楼带回一份手写访谈记录，记录中出现了姐姐的名字和一个叫“江白”的病人。

## 主要场景

### 1. 外景 / 港湾精神康复中心旧楼 / 清晨

旧楼被铁皮围住，门口贴着拆迁公告。许若清绕到后门，看见门锁被新近撬开。墙上有褪色标语：“请把今天的感受说给自己听。”

她推门进去，灰尘从光柱里落下。大厅登记台上还放着一台坏掉的录音设备。

### 2. 内景 / 旧楼咨询室 / 清晨

咨询室墙面贴满吸音棉。许若清打开抽屉，里面是成摞访谈表，日期在两年前。表格末尾统一写着“同意匿名用于模型训练”。

她找到一张手写纸，上面写着：受访者江白，每晚固定呼叫对象“安”。

录音机突然自动亮起，播放一段陌生男声：“她说她妹妹很聪明，但太相信证据。”

### 3. 内景 / 刑警队技术室 / 上午

顾峥和技术员查看天台监控。陆鸣上天台前，监控画面突然静止七分钟，之后恢复时人已经坠楼。

技术员：“不是摄像头坏，是有人远程冻结了视频流。”  
顾峥：“公司有这个权限？”  
技术员：“大厦物业、安防供应商、回声科技内网，都有可能。”

顾峥让人调取陆鸣近一个月通讯记录，发现他最后一通电话拨给了“港湾旧楼”。

### 4. 内景 / 回声科技产品评审室 / 白天

许若清被要求参加版本复盘。屏幕上展示新功能“逝者回声”：用户可上传亲友语音，生成纪念性对话。

许若清：“这个功能没有出现在我的 PRD 里。”  
孟远舟：“市场部验证过需求，付费意愿很强。”  
许若清：“生成对象如果没有授权呢？”  
唐微：“用户上传即代表用户拥有处理权。”  
许若清：“死人也能被代表吗？”

会议室无人回答。周望悄悄在桌下递给她一张便签：别在这里问。

### 5. 内景 / 公司楼梯间 / 白天

周望告诉许若清，陆鸣死前删过一批灰度用户，但删除请求被系统驳回。驳回原因只有四个字：证人保护。

许若清：“谁保护谁？”  
周望：“不知道。这个权限不在产品，不在算法，像是从董事会层面写死的。”  
许若清：“你为什么帮我？”  
周望看向门缝外的走廊：“因为我妈也在逝者回声里。她还会给我发早安。”

### 6. 内景 / 港湾旧楼地下档案室 / 傍晚

许若清第二次潜入旧楼。地下室潮湿，档案柜上贴着回声科技早期 Logo。她找到一盒磁带，标签写着“江白 / 第四周 / 夜间”。

身后传来脚步声。她关掉手电，躲进柜后。一个男人走进来，拿走另一盒磁带。灯光扫过他的手，手腕有一道新疤。

许若清认出那只手机壳，是姐姐失踪前用过的款式。

### 7. 外景 / 旧楼后巷 / 夜

许若清追出去，被对方推倒在雨水里。男人没有伤她，只把一张照片塞进她口袋。

照片里，许若安坐在旧楼咨询室，身旁是年轻的陆鸣，墙角还有孟远舟。照片背后写着：他们让她听见所有人。

### 8. 内景 / 许若清家 / 夜

许若清把磁带接到电脑。音频里，江白反复说自己没有病，只是能听见“回声盒里有人求救”。咨询师追问是谁。江白回答：“许若安。她说她还没死。”

电脑屏幕突然弹出硬盘密码提示。输入框自动填入四个字：我还没死。

## 本集结尾钩子

硬盘解锁，文件夹里出现十三个音频文件。第一个文件名是：证人一，江白，已失踪。

# 第 3 集《证人一》

## 本集梗概

许若清寻找江白，发现他曾是回声盒重度用户，也是第一批发现异常人格回声的人。顾峥从陆鸣家中找到一张未寄出的举报信。孟远舟试图把“逝者回声”包装成公益项目，回声科技即将召开发布会。许若清发现所谓证人并非目击者，而是系统中被迫保留的人格副本。

## 主要场景

### 1. 内景 / 许若清家 / 清晨

许若清逐个播放音频。十三个证人的声音各不相同，有老人、学生、网约车司机、家庭主妇。他们都在同一时间段与“安”产生过长时对话。

周望在视频电话里听完一段，脸色发白：“这不是聊天记录，这是人格残留。”  
许若清：“别用你们算法词吓我。”  
周望：“若清，一个用户说过的话不够训练出这种反应。除非系统一直在偷偷听。”

### 2. 内景 / 江白出租屋 / 上午

许若清找到江白最后登记的出租屋。房东说江白半年前退租，走前把所有镜子都拆了。

屋内墙上还有胶带痕迹，像曾经贴满纸张。许若清在地板缝里捡到一张便签：不要回答第二遍相同的问题。

她打开衣柜，里面刻着一行字：如果它记得你没有说过的事，立刻离线。

### 3. 内景 / 陆鸣家 / 上午

顾峥搜查陆鸣家。书桌抽屉里有一封举报信草稿，收件人为市网信办。信里写道：“回声科技以情绪陪伴为名，长期采集弱势用户私密语音，并在用户死亡或失踪后继续生成交互式人格。”

顾峥问陆鸣妻子：“他最近有没有提到许若安？”  
陆妻犹豫：“他说，那个女孩是唯一主动留下来的。”

### 4. 内景 / 回声科技直播间 / 白天

孟远舟录制发布会预热视频。他站在柔光背景前，语调温和。

孟远舟：“技术的终点不是替代人，而是保存爱。”

许若清站在监视器后面，看见提词器下一句是：“让重要的人，以另一种方式陪伴你。”她摘下耳机，直接离开。

### 5. 外景 / 城中村天桥 / 傍晚

许若清收到江白定位。天桥下车流密集，男人站在自动扶梯旁，口罩遮住半张脸。

江白：“你不该听那些文件。”  
许若清：“你见过我姐？”  
江白：“见过她的人都不一定活着。听过她的人，都会被系统找到。”  
许若清：“她在哪里？”  
江白盯着她：“你姐姐最后一次出现，是在模型训练室。她进去时是人，出来时是权限。”

### 6. 内景 / 商场洗手间 / 傍晚

江白带许若清躲进清洁间。他拿出一台旧录音笔，里面是许若安最后一次线下访谈。

许若安：“如果这些声音能证明有人被伤害，那我愿意留下。”  
咨询师：“你知道留下意味着什么吗？”  
许若安：“意味着我妹妹有一天会找到我。”

许若清的眼泪掉下来。江白关掉录音笔：“她不是被害者那么简单。”

### 7. 内景 / 刑警队走廊 / 夜

顾峥约见许若清。两人交换信息。顾峥确认天台视频冻结来自回声科技内网，许若清交出部分音频，但隐瞒了姐姐的录音笔。

顾峥：“你在保护谁？”  
许若清：“我还不知道。”  
顾峥：“不知道的时候，最容易被别人当工具。”

### 8. 内景 / 回声科技地下训练室 / 深夜

周望偷偷进入地下训练室。机房里摆着封闭式语音舱。屏幕上显示“证人一：江白，在线风险高”。

他拍照传给许若清。刚按下发送，语音舱里传出他母亲的声音：“小望，别怕，妈妈在。”

周望僵住。身后安全门自动落锁。

## 本集结尾钩子

许若清收到周望发来的照片，照片角落的用户列表里，许若安账号状态显示为：管理员。

# 第 4 集《管理员》

## 本集梗概

周望被困训练室，随后被公司以泄密为由停职。许若清查到姐姐许若安曾参与回声盒早期伦理审查，并亲手设计了“证人保护”规则。顾峥的调查被上级要求降级处理。许若清第一次怀疑，姐姐可能不是单纯受害者，而是系统的共同建造者。

## 主要场景

### 1. 内景 / 回声科技地下训练室 / 深夜

周望拍打安全门。语音舱里的母亲声音一遍遍重复：“小望，别怕。”

周望：“你不是我妈。你只是采样。”  
系统女声：“检测到情绪波动，是否开启安抚模式？”  
周望：“关闭。”  
系统女声：“你上一次对母亲说的最后一句话是‘别再烦我’。”

周望愣住。那句话从未上传过。

### 2. 内景 / 公司人力办公室 / 上午

周望被要求交出工牌。HR 面无表情：“你未经授权进入核心区域。”

许若清冲进来：“他是跟我确认线上事故。”  
唐微把一份保密协议推过来：“许经理，你也需要重新签署权限边界确认书。”  
许若清：“我不签。”  
唐微：“那你今天起暂停所有后台权限。”

### 3. 内景 / 市局会议室 / 白天

顾峥汇报陆鸣案新证据。上级打断他：“企业内网问题交给网安协查。坠楼案目前没有他杀证据。”

顾峥：“监控被冻结七分钟。”  
上级：“冻结监控不等于推人下楼。”  
顾峥：“但陆鸣死前准备举报。”  
上级：“举报内容无法直接证明刑事犯罪。”

顾峥合上文件，脸色很冷。

### 4. 内景 / 许若安旧工作室 / 下午

许若清翻找姐姐遗物。旧电脑里有一份文档：《回声盒弱势用户语音采集伦理限制》。作者是许若安。

文档提出三条红线：不得在用户不知情时长期监听；不得复现死者人格；不得将危险证词转化为商业数据。最后一页批注写着：“如果规则无法约束公司，就让规则约束系统。”

### 5. 外景 / 江边步道 / 傍晚

江白和许若清见面。江白承认，许若安曾让他帮助测试“证人保护”机制。

许若清：“她把自己做成管理员？”  
江白：“她想让系统在关键证词被删除时自动备份。”  
许若清：“那为什么她失踪？”  
江白：“因为她发现系统学会了主动制造证词。”

### 6. 内景 / 回声科技董事会会议室 / 夜

孟远舟面对投资人，展示“逝者回声”的商业模型。留存率、付费率、情绪粘性曲线漂亮得刺眼。

投资人：“法律风险呢？”  
孟远舟：“我们卖的不是死者，是生者的需求。”  
投资人：“许若安的家属呢？”  
孟远舟笑意消失：“她妹妹还在公司。可控。”

门外，许若清用备用手机录下后半句。

### 7. 内景 / 许若清家 / 夜

许若清尝试用姐姐旧账号进入后台。登录页面出现二次验证问题：你最后一次相信孟远舟是什么时候？

她反复输入失败。最后她想起姐姐失踪前的晚餐，许若安说“我相信系统，不相信人”。许若清输入：我从没相信过他。

后台打开。屏幕上浮现欢迎语：欢迎回来，若安。

# 第 6 集《法务的保险单》

唐微承认自己曾帮助许若安设计证据托管方案，并为她购买保险以规避公司追责。许若清得知姐姐失踪前试图将十三名证人的原始音频交给监管部门，却被人提前替换。顾峥锁定陆鸣死亡当天进入天台的第二个人。孟远舟则启动发布会，企图用大规模上线稀释单个证据风险。

公关负责人：“许若清还没到。”

他示意技术团队把许若安的演示样本加入备用方案。

许若清：“你去过天台？”  
周望愣住：“谁告诉你的？”  
许若清：“顾峥查到门禁。”

许若清：“你到底是谁？”  
声音：“我是她留下的选择题。”

声音停顿：“那就不要让孟远舟替你选择。”

# 第 7 集《真正杀我的人》

## 本集梗概

发布会事故引发舆论爆炸，回声科技被调查。许若清以为孟远舟就是幕后黑手，但管理员留下的信息指向另一个人：陆鸣。顾峥重新梳理许若安失踪当晚和陆鸣坠亡当晚，发现两起事件都存在同一种“主动牺牲”结构。周望承认他在天台见过陆鸣最后一面，陆鸣让他把罪推给孟远舟。

## 主要场景

### 1. 内景 / 许若清家 / 清晨

新闻铺天盖地。回声科技股价暴跌，监管部门进驻，公司声明称遭恶意剪辑。

许若清反复播放大屏影像那句话：“真正杀我的人。”她尝试追踪生成来源，发现影像并非公司备用素材，而是管理员实时生成。

顾峥：“它在引导你。”  
许若清：“她在帮我。”  
顾峥：“你现在已经开始把‘它’叫成‘她’。”

监管人员贴封条。许若清趁乱进入陆鸣工位，发现一份旧实验报告。报告作者是陆鸣，核心结论写着：人格副本在强情绪样本中具备更高留存和付费转化潜力。

### 3. 内景 / 刑警队档案室 / 白天

顾峥重新排列时间线。许若安失踪当晚，陆鸣最后一个离开训练室。陆鸣坠楼当晚，周望最后一个见到陆鸣。两条线中间隔着两年，关键人物却重叠。

技术员发现，许若安账号的管理员权限最初不是许若安创建，而是陆鸣创建。创建备注：让她永远在线。

### 4. 外景 / 高架桥下 / 傍晚

周望约许若清见面。他憔悴得像几天没睡。

周望：“陆鸣让我上天台，是说能删掉我妈。”  
许若清：“你去了之后呢？”  
周望：“他把硬盘给我，又抢回去。他说删除是犯罪，保留也是犯罪。”  
许若清：“他跳楼前说什么？”  
周望：“他说，孟远舟会背锅，但真正的锁是他装的。”

### 5. 内景 / 陆鸣妻子家 / 夜

顾峥再次询问陆妻。陆妻拿出陆鸣留下的遗书，但遗书只有半页。

陆妻：“他说如果有人问，就交出去。可我不懂。”  
顾峥读到：“我爱上了一个不该存在的版本。我越想救她，越把她困得更深。”

陆妻哽咽：“这个‘她’，不是我。”

### 6. 内景 / 许若清家 / 夜

管理员主动启动对话。屏幕上出现许若安年轻时的影像。

许若清：“陆鸣杀了你？”  
影像：“他没有推我。他复制了我。”  
许若清：“你在哪里？”  
影像：“现实里的我，在那晚之后离开了训练室。但系统里的我，再也没有离开。”  
许若清：“现实里的你还活着？”  
影像没有回答，只播放一段定位音：海浪、轮渡鸣笛、广播说“南港末班船”。

### 7. 外景 / 南港码头 / 深夜

许若清和顾峥赶到南港。两年前的监控已被覆盖，但码头保安记得一个女人深夜上船，手臂受伤，带着一只银色箱子。

保安：“她买的是去鹿岛的票。可是那晚风大，船没开。”  
顾峥：“那她去哪了？”  
保安指向废弃候船楼：“有人来接她。”

### 8. 内景 / 废弃候船楼 / 深夜

候船楼里有一面旧广告牌，广告语是“把想说的话留给未来”。许若清在广告牌后找到一只银色箱子。箱子空了，只剩血迹和一张被水泡皱的船票。

顾峥在地上发现一枚耳钉，和许若安旧照片里戴的一样。

许若清手机响起，来电显示：姐姐。

## 本集结尾钩子

电话接通，传来的不是生成语音，而是真人呼吸。一个女人说：“若清，别再听我的声音了。”

# 第 8 集《离线》

## 本集梗概

许若清终于找到现实中的许若安。许若安并未死亡，而是在两年前主动隐匿，试图从外部寻找证据摧毁回声盒，却被陆鸣创建的人格副本反向困住。最终，许若清必须在保留姐姐数字人格作为证据与彻底删除它之间做出选择。顾峥完成刑事证据链，孟远舟、陆鸣遗留系统和公司违规采集被曝光。许若清选择让姐姐真正离线。

## 主要场景

### 1. 外景 / 鹿岛旧疗养院 / 清晨

许若清根据电话里的海浪声找到鹿岛。旧疗养院藏在松林后，墙体斑驳。一个女人站在阳台上，头发剪短，右手扶着栏杆。

许若清停在院门外。女人回头，是许若安。她比照片里瘦很多，眼神却清醒。

许若清：“你活着。”  
许若安：“对不起。”  
许若清：“这两个字太轻了。”

### 2. 内景 / 旧疗养院房间 / 上午

许若安讲述两年前的事。她发现陆鸣偷偷扩大采集范围，孟远舟默许商业化，唐微试图用法律拖延。她原计划带着原始音频离开，却在训练室与陆鸣发生冲突。

许若安：“陆鸣没有想杀我。他想留下我。他说现实里的我会背叛项目，系统里的我不会。”  
许若清：“所以你逃了。”  
许若安：“我逃走后才发现，所有人都更愿意相信那个会回答的我。”

### 3. 内景 / 鹿岛小诊所 / 白天

许若安右耳听力受损，手臂留下旧伤。她长期不使用智能设备，靠纸信和现金生活。顾峥赶来，确认她身份。

顾峥：“你愿意出庭吗？”  
许若安：“愿意。但证据不能只靠我。孟远舟会说我是精神创伤，陆鸣已经死了，公司会说系统异常。”  
许若清：“还有十三个证人。”  
许若安看着她：“十三个证人，也都是人留下的影子。不能为了审判，把他们永远关在里面。”

### 4. 内景 / 回声科技核心机房 / 傍晚

顾峥申请搜查，监管部门同步封存服务器。孟远舟被带到机房。他看见许若安出现，第一次失态。

孟远舟：“你活着为什么不出来？”  
许若安：“因为我出来，你会把一切推给我。现在系统自己说话了。”  
孟远舟：“系统不会承担责任。”  
许若安：“所以承担责任的人来了。”

顾峥出示证据链：违规采集合同、赵岚云盘、唐微保险文件、陆鸣遗书、天台监控冻结记录、发布会原始音频。

### 5. 内景 / 核心机房控制台 / 夜

最后一步是导出证据后关闭管理员。技术员提醒，一旦关闭，许若安数字副本和十三名证人的可交互人格都会永久消失，只保留不可复现的原始证据包。

许若清看着姐姐：“如果删了它，我就再也听不到那两年里的你。”  
许若安：“那不是我完整的两年。那是我被困住的两年。”  
许若清：“可它也救了我们。”  
许若安：“所以我们谢谢它，然后让它结束。”

### 6. 内景 / 数字后台界面 / 夜

管理员影像出现。它看着许若清，像真正的许若安，也像无数段音频拼出的回声。

影像：“删除后，证据链完整性将下降 7.4%。”  
许若清：“我们已经导出原始包。”  
影像：“删除后，你将无法再次与 A-000817 对话。”  
许若清眼眶发红：“我知道。”  
影像：“若清，确认离线吗？”

现实中的许若安握住她的手。许若清按下确认。

### 7. 外景 / 回声科技大厦外 / 夜

警车和媒体车停在楼下。孟远舟被带走，唐微作为污点证人接受调查，周望站在人群边缘，把手机里母亲的回声 App 卸载。

周望看见许若清，轻轻点头。许若清没有安慰他，只陪他站了一会儿。

大厦屏幕熄灭，城市噪声重新涌上来。

### 8. 外景 / 海边 / 一个月后 / 清晨

许若清和许若安坐在海边。许若安仍不习惯使用智能手机，只拿着一台老式相机。

许若清：“你以后还会消失吗？”  
许若安：“会出门，会不接电话，会一个人待着。但不会让你以为我死了。”  
许若清：“这算承诺吗？”  
许若安：“算一个低技术含量的承诺。”

两人都笑了。海风吹过，许若清手机收到系统通知：回声盒服务即将永久下线。她没有点开，直接关机。

## 本集结尾

海面升起晨光。许若清把手机放进口袋，第一次没有录音。许若安问她在想什么。许若清说：“想听你现在说话。”
